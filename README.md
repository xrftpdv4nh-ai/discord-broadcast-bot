# Discord Broadcast Bot

بوت ديسكورد لإرسال الرسائل الجماعية لجميع أعضاء السيرفر في الخاص مع إحصائيات مفصلة.

## المميزات

- ✅ إرسال رسائل لجميع أعضاء السيرفر في الخاص
- 📊 إحصائيات مفصلة لحالة الإرسال
- ⚡ نظام حماية من الحظر مع تحكم في السرعة
- 🔒 نظام صلاحيات آمن
- 📈 تتبع حالة الإرسال في الوقت الفعلي
- 🛡️ معالجة شاملة للأخطاء

## التثبيت والإعداد

### 1. تثبيت Node.js
تأكد من تثبيت Node.js (الإصدار 16 أو أحدث) على جهازك.

### 2. تثبيت المتطلبات
```bash
npm install
```

### 3. إنشاء البوت في Discord Developer Portal

1. اذهب إلى [Discord Developer Portal](https://discord.com/developers/applications)
2. انقر على "New Application"
3. اختر اسماً للبوت
4. اذهب إلى تبويب "Bot"
5. انقر على "Add Bot"
6. انسخ الـ Token

### 4. إعداد ملف البيئة

انسخ محتويات `.env` وأدخل المعلومات المطلوبة:

```env
DISCORD_TOKEN=YOUR_BOT_TOKEN_HERE
CLIENT_ID=YOUR_BOT_CLIENT_ID_HERE
OWNER_ID=YOUR_DISCORD_USER_ID_HERE
```

### 5. دعوة البوت للسيرفر

استخدم هذا الرابط (استبدل CLIENT_ID بمعرف البوت):
```
https://discord.com/api/oauth2/authorize?client_id=CLIENT_ID&permissions=2048&scope=bot
```

الصلاحيات المطلوبة:
- Send Messages
- Embed Links
- Read Message History

## الاستخدام

### الأوامر المتاحة

#### `-send <message>`
إرسال رسالة لجميع أعضاء السيرفر في الخاص

**مثال:**
```
-send مرحباً بكم جميعاً في السيرفر الجديد!
```

#### `-show`
عرض إحصائيات البوت والسيرفر

#### `-help`
عرض قائمة الأوامر المتاحة

## إعدادات الأمان

البوت مصمم لتجنب الحظر من Discord من خلال:

- **تأخير بين الرسائل:** 2 ثانية (قابل للتعديل)
- **معالجة بالمجموعات:** 10 رسائل في كل مجموعة
- **تأخير بين المجموعات:** 30 ثانية
- **معالجة شاملة للأخطاء**

يمكن تعديل هذه الإعدادات في ملف `.env`:

```env
DM_DELAY=2000          # تأخير بين الرسائل (بالميلي ثانية)
BATCH_SIZE=10          # عدد الرسائل في كل مجموعة
BATCH_DELAY=30000      # تأخير بين المجموعات (بالميلي ثانية)
```

### التشغيل العادي


### التشغيل في وضع التطوير (مع إعادة التشغيل التلقائي)

## الإحصائيات المتاحة

عند استخدام أمر البرودكاست، ستحصل على:

- 👥 إجمالي عدد الأعضاء
- ✅ عدد الرسائل المرسلة بنجاح
- ❌ عدد الرسائل الفاشلة
- 📊 معدل النجاح
- ⏱️ الوقت المستغرق
- 📋 قائمة بالأعضاء الذين فشل إرسال الرسالة إليهم مع السبب

## أسباب فشل الإرسال الشائعة

- **الرسائل الخاصة مغلقة:** العضو أغلق الرسائل الخاصة
- **ليس لدى البوت صلاحية:** البوت محظور من قبل العضو
- **المستخدم غير موجود:** العضو غادر السيرفر أثناء الإرسال

## الأمان والصلاحيات

- فقط مالك البوت أو الأعضاء الذين لديهم صلاحية Administrator يمكنهم استخدام أمر الإرسال
- البوت لا يحفظ أي رسائل أو بيانات شخصية
- جميع العمليات تتم في الذاكرة المؤقتة فقط

## استكشاف الأخطاء

### البوت لا يرد على الأوامر
- تأكد من أن البوت متصل ولديه الصلاحيات المطلوبة
- تأكد من استخدام البادئة الصحيحة (`-`)

### فشل في إرسال بعض الرسائل
- هذا طبيعي، بعض الأعضاء قد يكون لديهم الرسائل الخاصة مغلقة
- راجع تقرير الإحصائيات لمعرفة الأسباب

### البوت بطيء في الإرسال
- هذا مقصود لتجنب الحظر من Discord
- يمكن تقليل التأخير في ملف `.env` لكن بحذر

## الدعم

إذا واجهت أي مشاكل:

1. تأكد من صحة إعدادات `.env`
2. تأكد من أن البوت لديه الصلاحيات المطلوبة
3. راجع ملفات السجل للأخطاء
4. تأكد من أن إصدار Node.js محدث

## تحذيرات مهمة

- **لا تشارك الـ Token مع أحد**
- **استخدم البوت بمسؤولية ولا ترسل رسائل مزعجة**
- **احترم قوانين Discord وقوانين السيرفر**
- **لا تقلل التأخير كثيراً لتجنب الحظر**
